# MLForge-Studio

**MLForge-Studio** is an open-source Python framework for **Machine Learning**, designed to make building, training, and evaluating models simple and intuitive.  
It supports **supervised learning, unsupervised learning, and deep learning** with a consistent API.

## 🚀 Features

- **Supervised Learning**: Linear Regression, Logistic Regression, KNN, and more  
- **Unsupervised Learning**: KMeans, PCA, etc.  
- **Deep Learning**: Simple neural networks with dense layers, activation functions, and backpropagation  
- **Gradient Descent**: Configurable `learning_rate` and `epochs` for iterative training  
- **Consistent API**: All models inherit from `BaseModel` with `fit()` and `predict()` methods  
- **Extensible**: Add new models, layers, or preprocessing methods easily
